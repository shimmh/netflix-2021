# MOVIETRIP

Netflix를 Clone 한 토이 프로젝트로 모든 영화, TV 프로그램을 한곳에서 찾아볼 수 있는 영화 정보 제공 사이트!

## 사용 기술

react, react hook, typescript, html, css, styled-component

## 활용 API

[The Movie Database API](https://developers.themoviedb.org/)

## Screens

- Home
- Movie
- TV Shows
- Search
- Detail

## API Verbs

### Movie

- Now playing (Movie)
- Upcoming (Movie)
- Popular (Movie)
- Movie Detail
- Search
- Similar Movie

### TV

- Top Rated (TV)
- Popular (TV)
- Airing Today (TV)
- TV Show Detail
- Search
- Similar Show
